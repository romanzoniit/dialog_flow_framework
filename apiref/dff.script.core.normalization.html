Normalization#

Normalization module is used to normalize all python objects and functions to a format that is suitable for script and actor execution process. This module contains a basic set of functions for normalizing data in a dialog script.

map_deprecated_key(key)[source]#

This function is used to map deprecated keyword to new one.

Parameters:

key (str) – A keyword of a node.

Return type:

str

Returns:

A mapped keyword of a node.

normalize_condition(condition)[source]#

The function that is used to normalize condition

Parameters:

condition (Callable) – Condition to normalize.

Return type:

Callable

Returns:

The function condition wrapped into the try/except.

normalize_keywords(script)[source]#

This function is used to normalize keywords in the script.

Parameters:

script (Dict[Union[str, Keywords], Dict[Union[str, Keywords], Dict[Keywords, Any]]]) – Script, containing all transitions between states based in the keywords.

Return type:

Dict[Union[str, Keywords], Dict[Union[str, Keywords], Dict[str, Any]]]

Returns:

Script with the normalized keywords.

normalize_label(label, default_flow_label='')[source]#

The function that is used for normalization of default_flow_label.

Parameters:

  • label (Union[Callable, Tuple[str, float], Tuple[str, str], Tuple[str, str, float], str]) – If label is Callable the function is wrapped into try/except and normalization is used on the result of the function call with the name label.

  • default_flow_label (Union[str, Keywords]) – flow_label is used if label does not contain flow_label.

Return type:

Union[Callable, Tuple[str, str, float]]

Returns:

Result of the label normalization, if Callable is returned, the normalized result is returned.

normalize_processing(processing)[source]#

This function is used to normalize processing. It returns function that consecutively applies all preprocessing stages from dict.

Parameters:

processing (Dict[Any, Callable]) – Processing which contains all preprocessing stages in a format “PROC_i” -> proc_func_i.

Return type:

Callable

Returns:

Function that consequentially applies all preprocessing stages from dict.

normalize_response(response)[source]#

This function is used to normalize response, if response Callable, it is returned, otherwise response is wrapped to the function and this function is returned.

Parameters:

response (Union[Message, Callable[..., Message], None]) – Response to normalize.

Return type:

Callable[..., Message]

Returns:

Function that returns callable response.

normalize_script(script)[source]#

This function normalizes Script: it returns dict where the GLOBAL node is moved into the flow with the GLOBAL name. The function returns the structure

{GLOBAL: {…NODE…}, …} -> {GLOBAL: {GLOBAL: {…NODE…}}, …}.

Parameters:

script (Dict[Union[str, Keywords], Any]) – Script that describes the dialog scenario.

Return type:

Dict[Union[str, Keywords], Dict[Union[str, Keywords], Dict[str, Any]]]

Returns:

Normalized Script.

normalize_transitions(transitions)[source]#

The function which is used to normalize transitions and returns normalized dict.

Parameters:

transitions (Dict[Union[Callable, Tuple[str, float], Tuple[str, str], Tuple[str, str, float], str], Callable]) – Transitions to normalize.

Return type:

Dict[Union[Callable, Tuple[str, str, float]], Callable]

Returns:

Transitions with normalized label and condition.
